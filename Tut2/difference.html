Class

  • Class is preceeded by a '.' in css
  • Class is given to an element that might repeat several times in a page
  • We can say Class is a type of an item

Id

  • Id is preceeded by a '#' in css
  • Id is given to an unique element
  • We can say Id is a name of an unique item

Note: There is an example of styling a class and an id in main.css
